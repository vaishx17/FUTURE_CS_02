# FUTURE_CS_02 – Phishing Email Detection & Awareness System

## Objective
Analyze phishing email samples and create a professional awareness report to help users identify and prevent phishing attacks.

## Tools Used
- Manual email content analysis
- Domain inspection techniques
- Awareness-based risk classification model
- MS Word (Documentation)

## Analysis Approach
Two phishing email samples were analyzed to identify:
- Suspicious sender domains
- Malicious or deceptive links
- Psychological manipulation tactics
- Risk level classification

## Key Outcome
The report demonstrates how phishing attacks use urgency, fear, and domain spoofing to deceive users, and provides structured prevention guidelines for organizations.
